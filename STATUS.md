## 关键架构决策

| 决策 | 原因 |
|------|------|
| 无 JWT，`userId` 走参数 | 简化鉴权，服务端 `assertSuperAdmin` 兜底 |
| `group_name` 存 JSON 数组（TEXT 列） | 不加表，兼容旧单值格式，`normalizeStudent` 统一解析 |
| 反馈图片客户端压缩到 ≤500KB | D1 单行有大小限制，Canvas 压缩后再存 Base64 |
| `notification_reads` 无 CASCADE | 删通知时需手动先删 `notification_reads` 再删 `notifications` |
| 经验值双轨 | `exp`（本宠，可扣）+ `lifetime_exp`（累积，只增），分别驱动等级和战力榜 |
